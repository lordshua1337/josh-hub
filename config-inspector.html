How config loading works: When you start a Claude Code session, it automatically loads your global CLAUDE.md, all rule files, and settings.json into context (~2,167 tokens). Everything else is on-demand: agents load when dispatched via Task tool, commands load when you type the slash command, and skills load when matched by name. Project CLAUDE.md files only load when you're in that project directory.
